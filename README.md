SkyPay Banking Service - Technical Test

This project is a Fullstack implementation of a banking system designed to manage transactions and generate account statements.

   Architecture
Backend: Spring Boot (Java 17)
Frontend: React.js
Data Management: In-memory (ArrayList) for optimal performance.

Features
Deposit and Withdrawal: Seamlessly add or remove funds.

Real-time Balance: View current account totals instantly.

Transaction History (Statement): Records sorted from newest to oldest.

Amount Validation: Prevents negative or invalid numbers from being processed.

Installation
Launch the Backend: Navigate to the /Backend folder and run the application.

Launch the Frontend: Navigate to the /frontend folder and run:

npm install
npm start
